hexo系统butterfly主题安装sakana组件的方法

发表于2022-11-01|更新于2022-11-01
|阅读量:

GitHub大佬写出了石蒜模拟器，本贴将展示如何把它添加到我们的博客上

效果展示


可以切换角色，点击最右边的X可以关掉（防止影响体验）

开整

在开始之前，请确保你有一个以hexo为系统，butterfly为主题（或者其他基于pug的主题）的博客

首先找到layout.pug
路径为：\themes\butterfly\layout\includes\layout.pug

接下来复制这段代码

#sakana-widget(style=’position:fixed;bottom:5px;right:5px;’)
script.
function initSakanaWidget() {
new SakanaWidget().mount(‘#sakana-widget’);
}
script(async=’’ onload=’initSakanaWidget()’ src=’https://cdn.jsdelivr.net/npm/sakana-widget@2.2.1/lib/sakana.min.js')

放到代码的最底下

没有读懂的可以看图，加了底色的是要添加的代码

保存后直接部署就可以了

补：第一行代码可以调整在屏幕上的位置（比如我习惯右手操作手机，在手机端上阻碍操作，所以改到了左边）

end
